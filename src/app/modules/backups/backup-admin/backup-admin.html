Respaldo y restauración

Copias de seguridad de la base de datos y restauración controlada

¿Qué incluye el backup actual?

Cada respaldo es un archivo .zip (infravial-full-backup-….zip) que contiene: manifest.json, database.json.gz (volcado BSON de MongoDB) y la carpeta uploads/ con todas las fotos de inventario y de catálogos del servidor.

Al restaurar un .zip, la aplicación vuelca la base de datos y reemplaza por completo la carpeta uploads/ del servidor por la del archivo (como desempaquetar). Los respaldos antiguos solo-BD (.json.gz) siguen funcionando: restauran MongoDB pero no tocan las fotos.

Restaurar desde tu PC (como importar Excel)

Preferible un .zip completo. También acepta .json.gz heredado (solo base de datos, sin tocar fotos).

Limpiar base de datos (irreversible)

Marca solo lo que quieres borrar. Se recomienda un backup antes. Los usuarios: se eliminan todos excepto tu sesión actual.
